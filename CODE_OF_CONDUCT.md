# Code of Conduct

This project follows the [Contributor Covenant v2.1][covenant] for
all interaction in issues, pull requests, discussions, and any
private communication that comes out of the project.

In short: be kind, be specific, assume good faith, don't expect
free support to be infinite. Disagreement is fine; harassment is
not.

## Reporting

If you experience or witness behaviour that violates the Covenant,
report it to the maintainer (handle on the GitHub profile, or via
GitHub's "Report content" workflow). Reports are kept confidential.
The maintainer will respond and take whatever action is
proportionate, up to and including blocking the offender from the
repo.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
